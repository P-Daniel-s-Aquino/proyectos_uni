# Proyectos Universitarios de Programación

¡Hola! 👋 Bienvenido a mi repositorio de proyectos universitarios. Este espacio es mi bitácora personal de aprendizaje y práctica en diversas tecnologías y lenguajes de programación. Aquí documento los ejercicios, trabajos prácticos y pequeños proyectos que desarrollo durante mis estudios.

## 🚀 Tecnologías y Lenguajes

Este repositorio contiene proyectos en los siguientes lenguajes:

*   **☕ Java:** Ejercicios enfocados en los pilares de la Programación Orientada a Objetos.
*   **🐍 Python:** Scripts y proyectos que exploran la versatilidad del lenguaje.
*   **⚙️ C++:** Prácticas de bajo nivel y gestión de memoria.
*   **🗃️ SQL:** Scripts y diseños de bases de datos relacionales.

## 📂 Estructura del Repositorio

El repositorio está organizado en carpetas, una por cada lenguaje o tecnología principal.

*   `SQL/`: Contiene scripts `.sql` con definiciones de tablas, consultas y trabajos prácticos de bases de datos.
*   `java_projects/`: Proyectos en Java, organizados por concepto de POO.
    *   `encapsulamiento/`: Ejercicios que demuestran la ocultación de datos.
    *   `herencia/`: Ejemplos de jerarquías de clases y polimorfismo.
*   `python_projects/`: Proyectos y scripts en Python.
*   `cpp_projects/`: Proyectos y ejercicios en C++.
*   `snippets_java/`: Una colección de mis snippets personales para VS Code que agilizan mi desarrollo en Java.

## 🎯 Ejemplos Destacados

Algunos de los proyectos clave que encontrarás aquí son:

1.  **Catálogo Musical (Java):** Una aplicación de consola que gestiona una colección de canciones, implementada con clases abstractas, herencia y polimorfismo para modelar distintos tipos de canciones.
2.  **Simulador de Cerradura (Java):** Un ejercicio clásico de encapsulamiento que simula el comportamiento de una cerradura de seguridad con intentos limitados.
